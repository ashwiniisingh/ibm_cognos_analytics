## 📌 Overview

This project analyzes how **social media usage impacts mental health and academic performance** using **IBM Cognos Analytics**.

It transforms raw data into **interactive dashboards** to uncover patterns in:
- Screen time  
- Sleep habits  
- Stress & anxiety levels  

---

## 🎯 Objectives

- 📊 Build structured **data models**  
- 📈 Design **interactive dashboards**  
- 🧠 Analyze **behavior vs mental health**  
- ⚙️ Apply Cognos features (filters, calculations)  

---

## 🗂️ Dataset Details

| Feature Category | Attributes |
|----------------|----------|
| 👤 Demographics | Age, Gender |
| 📱 Usage | Daily usage, Platform |
| 😴 Lifestyle | Sleep hours, Screen before sleep |
| 📚 Performance | Academic performance |
| 🧠 Mental Health | Stress, Anxiety, Addiction, Depression |

---

## 📈 Key Insights

- 🔥 High usage (300+ mins) → **Higher stress & anxiety**  
- 😴 Poor sleep → **Lower academic performance**  
- 📱 Multi-platform users → **Higher addiction levels**  
- 🤯 Social interaction ≠ **mental stability**
